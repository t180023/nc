  3. 発展問題 (余裕のある人だけやること)

    $\sqrt{1+x}$ のマクローリン展開を用いて$\sqrt{2}$ の近似値を計算するためには、$ x = 1$とすればよいが、収束半径が 1 であるため項数を増やしてもなかなか収束しない。実際、以下のようになる。

    n =         10 のとき sqrt(2) = 1.41920471191406  誤差：4.991150e-03 
n =        100 のとき sqrt(2) = 1.41435620590369  誤差：1.426435e-04 
n =       1000 のとき sqrt(2) = 1.41421802770420  誤差：4.465331e-06 
n =      10000 のとき sqrt(2) = 1.41421370343634  誤差：1.410632e-07 
n =     100000 のとき sqrt(2) = 1.41421356683344  誤差：4.460341e-09 
n =    1000000 のとき sqrt(2) = 1.41421356251408  誤差：1.409868e-10 
n =   10000000 のとき sqrt(2) = 1.41421356237748  誤差：4.388934e-12 
n =  100000000 のとき sqrt(2) = 1.41421356237317  誤差：7.327472e-14 
n = 1000000000 のとき sqrt(2) = 1.41421356237303  誤差：-6.195044e-14
    4の課題と同じように、$\displaystyle\sqrt{\frac{a+bx}{c+dx}}$ の形の関数のマクローリン展開を用いることによって、項数100以下で$\sqrt{2}$の有効数字15桁の近似値が得られるような $a, b, c, d$ の値を探しだせ。(注：もちろん、$ x = 1$ 以外の値を代入する。答は一つとは限らないが、なるべく簡単な $a, b, c, d$ の組み合わせを探すこと。)
