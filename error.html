エラーが出たときの対処法


  1. cc: kadai1-2.c: そのようなファイルやディレクトリはありません。
cc: no input files
    考えられる原因
    • 編集中のプログラムが保存されていない。
      プログラムを保存せよ!
    • 編集中のプログラムがどこか別のディレクトリにある。
      別のディレクトリに移動せよ! 
  2. kadai1-2.c: In function 'main':
kadai1-2.c:5: error: 'i' undeclared (first use in this function)
kadai1-2.c:5: error: (Each undeclared identifier is reported only once
kadai1-2.c:5: error: for each function it appears in.)
    考えられる原因
    • 変数を宣言し忘れている。( undeclared のメッセージあり。上記の場合はプログラム中5行目の変数 i )
      変数を宣言せよ! (上記の場合は5行目の i よりも前で) 
  3. kadai1-2.c: In function 'main':
kadai1-2.c:8: error: expected ';' before '}' token
    考えられる原因
    • セミコロン(;) を忘れている。( expected ';'のメッセージあり。)
      セミコロンを付けよ! 
  4. Undefined symbols:
  "_prinf", referenced from:
      _main in cciYLN8w.o
ld: symbol(s) not found
collect2: ld returned 1 exit status
    考えられる原因
    • スペルミス ( Undefined symbolsのメッセージあり。上記の場合は printf を prinf と間違えている。)
      スペルミスを直せ! 
  5. kadai1-2.c:5:10: warning: missing terminating " character
kadai1-2.c: In function 'main':
kadai1-2.c:5: error: missing terminating " character
kadai1-2.c:6: error: expected expression before '}' token
kadai1-2.c:6: warning: format not a string literal and no format arguments
kadai1-2.c:6: error: expected ';' before '}' token
    考えられる原因
    • ダブルクォート(")の閉じ忘れ ( missing terminating " characterのメッセージあり。)
      ダブルクォートを追加せよ! 
  6. kadai1-2.c: In function 'main':
kadai1-2.c:10: error: redeclaration of 'i' with no linkage
kadai1-2.c:5: error: previous declaration of 'i' was here
    考えられる原因
    • 同じ変数を2回宣言している。(redeclarationのメッセージあり。)
      余分な宣言を削除せよ! 
  7. kadai1-2.c: In function 'main':
kadai1-2.c:10: error: stray '\343' in program
kadai1-2.c:10: error: stray '\200' in program
kadai1-2.c:10: error: stray '\200' in program
    考えられる原因
    • 全角スペースがプログラムに混入している。
