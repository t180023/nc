    1. 上のデータをコピーし、data.txt という名前のファイルに保存せよ。ファイルの保存場所はディレクトリ nc とせよ。そして以下のプログラムを実行し、このテストの平均点を求めよ。 

      #include <stdio.h>

int main ()
{
  FILE *fp;
  fp = fopen("data.txt", "r");

  int x;
  double s = 0.0;
  int i = 0;

  while (fscanf(fp, "%d", &x) != EOF ) {
    s = s + x;
    i++;
  }

  printf("平均値は %g 点です。\n", s/i);

  fclose(fp);
}
      平均値は 67.763 点と出力されるはずである。

    4. 平均が $\mu$、標準偏差が $\sigma$ である分布に対して、チェビシェフの不等式 \[ P(|X -\mu| \ge k\sigma) \le \frac{1}{k^2} \] が成り立つ。これは平均値から標準偏差の $k$ 倍以上はなれている割合が全体の $\displaystyle\frac{1}{k^2}$ より小さいことを示している。$k=2$ とするとき、このデータに対して確かにチェビシェフの不等式が成り立っていることを確認せよ。
